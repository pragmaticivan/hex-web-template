About

Welcome to Hex.pm, the Erlang community's package hosting service. The purpose of this project is three-fold:

  1. Provide a better API for dealing with gems
  2. Create more transparent and accessible project pages
  3. Enable the community to improve and enhance the site

The project was started in January 2014 by Eric Meadows-Jönsson, and has since grown to include the contribution of over over 35 Programmers and millions of package downloads.

Some insights into the technical aspects of the site: It's 100% Elixir. The main site and API is a Phoenix application. Packages are hosted on Amazon S3, and the time between publishing a new package and having it ready for installation is minimal. For more info, please check out the code, which is APACHE licensed over at GitHub.
